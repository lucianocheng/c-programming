ESE 113/116 Lab: Struct Exercises

The purpose of this lab is to create a simple bowling database using structs and arrays.

Goals:

  • Define, declare and initialize C Structs
  • Access and modify member variables in structures
  • Use structures to organize data

C Structs: Background

Exercise: Bowling Database

Download and complete bowling.c. Extended instructions for completion are included in comments in the file. Once completed, this program will take in data for a bowling database as arguments, store it in an array of structs, and printf out the database when done.

Specific sections of code have been left blank for you to fill in. See the instructions in the file.

An example run might look like this (to demonstrate functionality, not any required formatting of output):

$ ./bowling Leela 10 20
