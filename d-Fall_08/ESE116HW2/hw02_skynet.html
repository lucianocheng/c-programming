Overview:

As promised in the previous lab, you will now be building SkyNet. Commissioned by DARPA and overseen by Microsoft, SkyNet is a comprehensive computer-based defense system. It is designed to remove the possibility of human error and slowness of reaction time to guarantee fast, efficient response to enemy attack.

Detailed function descriptions are in the function headers in skynet.c, please fill out the functions according to the specifications.

The file skynet.c contains 5 functions that you will have to complete.
Detailed specifications are in comments at the top of each function:

  • The count_down() function
    • The purpose of this function is to begin the countdown of the SkyNet initiation protocol
  • The get_human_max_age() function.
    • The purpose of this function is to calculate the maximum age of a human, as inputted by the user
  • The get_human_average_age() function.
    • The purpose of this function is to calculate the average age of the human ages inputted by the user
  • The calculate_resistance() function.
    • The purpose of this function is to calculate the expected human resistance, and the response the humans will send.
  • The query_user() function.
    • The purpose of this function is to query the user for a menu option
    • You are required to use a switch statement to complete this function.

The file skynet_main.c contains four functions you must utilize to write the skynet.c functions. Do not modify these functions, you must use them in their current form:

  • float get_float_from_terminal()
    • Retrieves one float value from the terminal, and returns this float
    • This function should only be used in the get_human_average_age() function
  • int get_integer_from_terminal()
    • It retrieves one integer value from the terminal, and returns this integer
  • char get_character_from_terminal()
    • It retrieves one character value from the terminal, and returns this character
  • void print_countdown_int(int print_int)
    • Prints a single countdown integer in the format "Countdown: %d\n".
    • Be sure to use this function in your count_down() function

The following definitions are in skynet.h:

  • For the calculate_resistance() function:
    • CARE_BEARS is defined as 0
    • TONY_STARK is defined as 1
    • BATMAN is defined as 2
    • CHUCK_NORRIS is defined as 3
  • For the query_user() function:
    • MENU_COUNTDOWN is defined as 0
    • MENU_MAX_AGE is defined as 1
    • MENU_AVG_AGE is defined as 2
    • MENU_RESISTANCE is defined as 3
    • MENU_HELP is defined as 4
    • MENU_EXIT is defined as 5
    • MENU_ERROR is defined as 6

Remember to use the above #defines when told to do so, DO NOT use the literal integer (0, 1, etc)

To compile your code to a binary executable called 'skynet', type in the following at the command line, assuming your skynet.c, skynet_main.c and skynet.h files are in your current working directory:

    gcc -Wall skynet_main.c skynet.c -o skynet

The '-Wall' flag will print out all warnings that the compilation generates. It is highly recommended that you eliminate warnings in your coding, as warnings can often lead to undefined behavior in your programs.

After executing the code above, the following line can be used to execute your program:

   ./skynet
