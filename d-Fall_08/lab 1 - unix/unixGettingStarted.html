  3. Getting Started with Emacs:

    Emacs is a powerful and popular program that we will be using to code C. Although Emacs has many functions, we will be using it as an editor.

    There won't be any questions about the emacs editor on the quiz. However, we will use emacs to write C programs during lab for the remainder of the semester.

    To those who use vi / vim as an editor, we recommend that you to take this opportunity to become familiar with emacs. If you stay in enginnering, it is highly likely that you'll need to emacs on the job, in a research lab, and/or in upper level classes.

    • To start Emacs, open a terminal (if you haven't already) and type "emacs &" at the prompt:
      student@computername:~> emacs &
      • The "&" tells the shell to launch emacs and let the shell continue to accept commands. Without "&", the shell waits for emacs to quit.
    • To open a new file in emacs:
      • Go to (File->Visit New File), or hold down the Control key and hit 'x' then 'f'
      • At the bottom of the window, it will ask you for a new file name (it might say "find file", this is normal).
        • Type "hello.c" and hit Enter.
        • If you get stuck at this stage, hit Escape three times to start over
    • Copy the following code and paste it into your new hello.c document (or buffer, as it is referred to in Emacs):
            #include <stdio.h>
      
      int main(int argc, char** argv){
         printf("Hello World!");
      }
      • You can use (Edit->Paste) to paste the code, or hold down the Control key and hit 'y'
    • Save the file by using (File->Save), or hold down the Control key and hit 'x' then 's'
      • When coding, remember to save frequently to avoid loss of data
    • The code should show up as different colors. If it is not, go to (Options->Show Syntax Highlighting)
    • Exit emacs by going to (File->Exit), or hold down the Control key and hit 'x' then 'c'
    • Open up the file again by typing the following at the command prompt:

      student@computername:~> emacs hello.c &
      The "hello.c" tells emacs to open up your file.
    • Exit Emacs to finish up
